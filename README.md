# globs-protobuf

Small library to serialize/deserialize Globs (from globsframework) to/from Protocol Buffers.

## Quick start

Add the dependency to your Maven project:

```xml
<dependency>
  <groupId>org.globsframework</groupId>
  <artifactId>globs-protobuf</artifactId>
  <version>4.0.0</version>
</dependency>
```
